How to run

PostgreSQL:-

Create DB: shipgen
Ensure user/password match current connection in backend/app/database.py:
postgresql+psycopg://postgres:postgres@localhost:5432/shipgen



Backend:-

cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m alembic -c alembic.ini upgrade head
uvicorn app.main:app --reload --port 8000


Frontend:-

cd frontend
npm install
npm run dev


Offline learning cycle (deterministic):-

curl -X POST http://127.0.0.1:8000/ml/offline-cycle ^
  -H "Content-Type: application/json" ^
  -H "Authorization: Bearer <admin_token>" ^
  -d "{\"runLabel\":\"weekly-01\"}"
